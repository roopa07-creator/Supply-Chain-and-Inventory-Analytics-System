# Supply Chain & Inventory Analytics System

## Overview
End-to-end analytics project built using SQL Server, SSIS, and SSRS to analyze products, orders, suppliers, and inventory data.

## Tech Stack
- SQL Server
- SSIS (ETL)
- SSRS (Reporting)
- GitHub

## Features
- Product, Orders, Supplier, and Inventory tables
- SSIS ETL pipelines with data validation
- SSRS reports with joins and calculations
- Order revenue and inventory tracking

## Reports
- Product Report
- Orders Report
- Inventory Report
